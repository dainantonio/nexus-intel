# Nexus Intel Dashboard

A cyberpunk-inspired intelligence dashboard for tracking AI, data, energy, and policy news.

## Features

- **Real-time News Feed**: Aggregates news from multiple sources
- **Interactive Map**: Visual geospatial intelligence
- **AI Analysis**: Gemini AI integration for insights
- **Bulk Operations**: Mass mark read/bookmark functionality
- **Multiple Views**: Row and grid layout options
- **Smart Search**: With recent searches and filtering
- **Cyberpunk Mode**: Optional visual effects
- **Keyboard Shortcuts**: Full keyboard navigation
- **Mobile Responsive**: Works on all screen sizes

## Setup

1. Clone or download the project files
2. Ensure all three files are in the same directory:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in a modern web browser

## Configuration

1. Click the settings gear icon
2. Configure API keys for:
   - Gemini AI (for enhanced analysis)
   - NewsAPI/GNews (for real news)
3. Adjust visual settings:
   - Cyberpunk mode
   - Default view mode
   - Accent color
4. Set up keyword filters

## Keyboard Shortcuts

- `Cmd/Ctrl + K`: Search
- `V`: Toggle view mode
- `R`: Refresh data
- `S`: Open settings
- `B`: Toggle bulk mode
- `M`: Mark focused item read
- `Shift + M`: Mark all visible read
- `ESC`: Close modals

## Dependencies

- Lucide Icons (loaded from CDN)
- Leaflet.js (for maps)
- ApexCharts (for data visualization)
- Font Awesome (icons)
- Google Fonts (Inter, JetBrains Mono)

## Browser Support

Works best in modern browsers (Chrome 90+, Firefox 88+, Edge 90+, Safari 14+)
